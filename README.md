# Simple Python Calculator

This is a console-based calculator built in Python.  
It supports basic arithmetic operations like addition, subtraction, multiplication, and division.  
The project is created as a mini-project for the course **Computer Science Fundamentals & Career Pathways (ETCCCP105)**.


## Features

- Perform **Addition**, **Subtraction**, **Multiplication**, and **Division**
- Simple **text-based menu**
- Handles **invalid input** (e.g., non-numeric values)
- Prevents **division by zero**
- Lets the user **perform multiple calculations** in a loop
- Clean and modular code using **separate functions**



##  Technologies Used

- **Language**: Python
- **Editor**: Visual Studio Code
